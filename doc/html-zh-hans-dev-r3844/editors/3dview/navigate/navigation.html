视图浏览¶

可以通过鼠标和键盘快捷键实现在3D空间中切换视图。

为了更好的在Blender的三维空间中工作，用户必须能够调整视角以及场景的观看方向。虽然将这些内容放在 3D视图 编辑器章节，但大多数其他编辑器都有类似的功能。 例如,可以在UV/图像编辑器中进行移动和缩放。

Tip

鼠标按钮和数字键盘

如果鼠标按键不足三个或者键盘没有小键盘区，请参阅 键盘和鼠标 页面，学习如何在Blender中使用它们。

旋转¶

参考

模式: 所有模式
菜单: 视图 ‣ 视图切换 ‣ 视轨
快捷键: MMB, Numpad2, Numpad4, Numpad6, Numpad8, Ctrl-Alt-Wheel

围绕着兴趣点旋转视图。在3D视图主区域范围内单击并拖动 MMB。 如果从视图区域的中间开始上下或左右移动, 那么视图就会绕着视图主区域中间旋转。

可以使用 Numpad8 和 Numpad2 (对应于垂直拖动 MMB)，或者使用 Numpad4 和 Numpad6 (或 Ctrl-Alt-Wheel)绕当前的视角全局 Z 轴旋转场景，按离散步长调整视角。 最后，还可以使用 Numpad9 切换到视图的另一侧。

另外，如果在 用户首选项 勾选了 模拟3键鼠标 选项，在视图主区域范围内按住 Alt 并拖动 LMB 模拟鼠标中键。

Note

快捷键

请记住，大多数快捷键影响 活动 区域 (有焦点的区域), 所以在使用快捷键之前，请检查鼠标光标是否位于要进行操作的区域.

See also

  • 旋转风格设置
  • 自动透视设置

横滚¶

参考

模式: 所有模式
菜单: 视图 ‣ 视图切换 ‣ 左倾/右倾
快捷键: Shift-Numpad4, Shift-Numpad6, Ctrl-Shift-Wheel

以15度的离散步长围绕其自身 Z 轴旋转摄像机视图。

平移¶

参考

模式: 所有模式
菜单: 视图 ‣ 视图切换 ‣ 平移
快捷键: Shift-MMB , Ctrl-Numpad2, Ctrl-Numpad4, Ctrl-Numpad6, Ctrl-Numpad8

上、下、左、右移动视图。在3D视图中按住 Shift 键拖动 MMB 平移视图。 与旋转一样，使用快捷键 Ctrl-Numpad8, Ctrl-Numpad2, Ctrl-Numpad4 and Ctrl-Numpad6 按照离散步长移动(注意: 你可以用 Shift 代替 Ctrl)。

对于那些没有鼠标中键的用户, 可以按住 Shift-Alt 同时拖动 LMB。

缩放¶

参考

模式: 所有模式
菜单: 视图 ‣ 视图切换 ‣ 放大/缩小
快捷键: Ctrl-MMB, Wheel, NumpadPlus, NumpadMinus

前后移动摄像机，可以按住 Ctrl 并拖动 MMB 来放大和缩小。快捷键是 NumpadPlus 和 NumpadMinus。 视图 ‣ 试图切换 子菜单也包含这些功能。更多信息，请参考上文3D视图的 视图 菜单图片。如果用的是滚轮鼠标，也可以通过 Wheel 来缩放。

Hint

如果你转晕了

如果你在3D视图中转晕了，这并不少见。两个快捷键可以帮到你: Home 更改视图以便您可以看到场景中的所有对象 视图 ‣ 查看全部。另外当处于透视模式时， NumpadPeriod 缩放视图到当前选择对象 视图 ‣ 查看所选。

缩放区域¶

参考

模式: 所有模式
菜单: 视图 ‣ 缩放区域
快捷键: Shift-B

缩放区域 工具用于指定一个矩形区域并放大，使该区域填充3D视图。

可以通过 视图 菜单或者快捷键 Shift-B 来访问，然后 LMB 单击并拖动选择需要缩放的矩形区域。

或者，你可以使用 MMB 拖拽出一个矩形，缩放框选区域。

滑动缩放¶

参考

模式: 所有模式
快捷键: Ctrl-Shift-MMB

在大多数情况下，它足以放大视图以便更近距离地观察某些东西, 然而, 你可能会注意到，到某一个点之后，无法继续拉近距离.

这是因为Blender存储了一个用于旋转和缩放的视点。它在很多情况下都很好，但是有时要将视点移到其他地方。这就是Dolly所支持的，将视图从一个位置移动到另一个位置。

可以通过按住 Ctrl-Shift 并拖动 MMB 来回滑动。
