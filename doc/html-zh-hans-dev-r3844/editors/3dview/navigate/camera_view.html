摄像机视图¶

参考

模式： 所有模式
菜单: 视图 ‣ 摄像机
菜单: 视图 ‣摄像机 ‣ 活动的摄像机
快捷键: Numpad0

猴头的摄像机视图。

摄像机视图显示当前的场景， 从当前的活动相机的角度来看. 它可以通过快捷键 Numpad0 来激活. 摄像机视图可以用来拍摄照片，并预览拍摄时场景的样子. 渲染的图像将包含虚线内的所有内容。在这个视图中，您还可以设置呈现 渲染区域 它定义要呈现的3D视图的部分。

摄像机视图为最终渲染提供预览。¶

摄像机视图。

渲染图像。

相机导航¶

有几种不同的方法可以在你的场景中导航和定位摄像机，其中一些解释如下。

在这个视图中，放大和缩小是可能的, 但是为了改变视角，您必须移动和旋转摄像机。

Note

请记住，活动的“摄像机”可以是任何类型的对象。因此，这些动作可以被用来定位和瞄准一盏灯。

将活动相机移动到视图¶

参考

模式: 物体模式
快捷键： CTRL-ALT-Numpad0

该功能用于使活动相机的位置和朝向匹配当前视图。

选择一个摄像机，然后在3D视图中移动到您想要的位置和你的相机的方向 (这样你就可以看到你想要看到的东西). 通过快捷键 Ctrl-Alt-Numpad0 和你选择的相机位置来匹配这个视图，然后切换到相机视图。

摄像机视图定位¶

通过 锁定到视图 在属性区域的视图面板中查看, 在相机视图中,你可以像平常一样在3D视图中导航, 同时保留在摄像机视图中。 控制和通常在3D视图中移动的情况是一样的。

旋转， 平移, 移动, 路径¶

为了执行这些摄像机的移动, 摄像机必须首先被 选择 中，这样它就变成了活动的对象 (当你通过它查看时，你可以:kbd:RMB -- 点击固定的矩形边框来选择它). 下面的操作也假设您在摄像机视图 Numpad0! 中。这样做之后, 你就可以像操作任何对象一样来操纵摄像机了。

滚动
按 R 进入物体旋转模式. 默认情况下，它将绕自身Z轴旋转 (与摄像机视图垂直的轴)旋转摄像机, 这就是相机 "roll"的定义。
垂直平移 或 倾斜
这是沿X轴的旋转. 按 R`进入对象旋转模式, 然后 :kbd:`X 两次 (第一次按下选择 全局 轴, 第二次按同样的字母选择 局部 轴 -- 这与任何轴都有效;查看 轴向锁定).
水平平移 或 Yaw
这相当于绕着相机的局部Y轴旋转. 按 R, 然后 Y 两次.
滑动
对于摄像机, 按 G 然后 MMB (或 Z 两次).
横向跟踪
按下 G`并移动鼠标(你可以使用 :kbd:`X 两次 或 Y 两次来获得纯水平或纯垂直的横向跟踪).
