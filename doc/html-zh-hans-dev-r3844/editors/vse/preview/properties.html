属性¶

场景预览/渲染¶

控制 场景条 中的图片在预览窗口的显示方式。

OpenGL预览

使用OpenGL快速预览 (更多内容参考 OpenGL 渲染 )，否则将使用较慢的完整渲染。

序列预览着色
OpenGL渲染方式，见3D视图 视图着色 选项。
实色纹理
在实体模式下同样显示纹理。

OpenGL预览使用的设置：

  • 活动窗口的抗锯齿用于所有场景。
  • alpha设置使用对应场景的 “alpha模式” 选项。

视图设置¶

显示曝光过度
使用斑马纹显示曝光过度(过白)区域。滑块用于调整阈值。
代理渲染尺寸
预览状态下代理显示的尺寸。小尺寸可以加快预览速度。

安全区¶

显示用于放置元素的参考线，以确保在所有屏幕中均可显示视频最重要的部分。

See also

见相机文档中 安全区 。

在预览窗口使用 蜡笔 。
