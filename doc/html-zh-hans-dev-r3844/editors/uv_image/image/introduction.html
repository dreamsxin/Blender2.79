控制¶

图像

数据块菜单 用于选择图像。在UV/图像编辑器载入或创建图像后， 属性区块 将出现图像面板。见 图像设置。

  • 渲染结果
  • 预览节点
钉住图像
忽略所选，只显示当前图像。

多图层¶

在UV/图像编辑器显示渲染图像时，会出现几个新的菜单选项。

槽
在渲染前选择新的槽，可以将连续多次渲染结果保存到渲染缓冲区。如果图像被渲染到了一个槽位，选中该槽即可查看图像。空白曹伟在UV/图像编辑器显示为空白网格。使用 the J 和 Alt-J 向前和向后循环查看渲染结果。
渲染层
使用 渲染层 是，该菜单用于选择显示的图层。
渲染通道
使用 渲染层 是，该菜单用于选择显示的通道。

通道¶

绘制通道

单选按钮用于设置显示的图像通道。

RGBA
显示透明像素为棋盘格背景，指代alpha通道。
RGB
绘制彩色图像，不显示alpha通道。
Alpha
显示alpha通道为灰阶图像。白色区域为不透明，黑色区域标示alpha为0。
Z缓冲
显示相机景深，范围为 相机设置 中指定的裁剪起始距离到结束距离。
红, 绿, 蓝
显示单通道为灰阶图像。

主视图¶

在按下 LMB 拖动时，将显示游标下方的颜色、游标位置和RGBA、HSV和亮度 色彩空间 表示的颜色值。
