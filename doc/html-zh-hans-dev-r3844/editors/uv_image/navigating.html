视图浏览¶

2D视图¶

按住 MMB 并拖动进行平移。

前后滚动鼠标滚轮 Wheel 进行缩放。 同样的，和3D视图一样，可以使用 NumpadPlus 或 NumpadMinus 缩放。

视图菜单¶

自动更新
同步更新其他视图窗口。
局部UV视图
ToDo.
显示其他物体
背景显示(物体模式)选中物体的UV。
显示元数据
显示在渲染标签页 元数据 设置的元数据。
Draw Texture Paint UVs
在图像绘制模式显示/隐藏UV。
放大/缩小视图
使用 Wheel 调整缩放。
缩放比例
  • 缩放 1:8 Numpad8
  • 缩放 1:4 Numpad4
  • 缩放 1:2 Numpad2
  • 缩放 1:1 Numpad1
  • 缩放 2:1 Shift-Numpad2
  • 缩放 4:1 Shift-Numpad4
  • 缩放 8:1 Shift-Numpad8
查看中心
查看全部UV，快捷键 NumpadPeriod。
查看全部
查看完整图像，快捷键 Home。
View Fit
调整图像显示尺寸至匹配视图，快捷键 Shift-Home。
