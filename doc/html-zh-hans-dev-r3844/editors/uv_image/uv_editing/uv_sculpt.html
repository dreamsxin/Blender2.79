UV雕刻¶

参考

模式: 绘制模式和遮罩模式
面板: 工具栏 ‣ 选项 ‣ UV雕刻, 工具栏 ‣ 工具
菜单: UV ‣ 雕刻t
快捷键: Q

UV雕刻 “模式” 就如同雕刻模式，可以抓起、拿捏和平滑UV。可以通过快捷键 Q 或者在 UV 菜单勾选UV雕刻激活该模式。

UV雕刻¶

激活UV雕刻后，工具栏会显示笔刷工具选择菜单和控制选项。

锁定边界
锁定UV孤岛边界，免受笔刷影响，用于保持UV孤岛形状。
雕刻所有孤岛
雕刻开始时，编辑所有孤岛，而非仅影响离笔刷中心最近的孤岛。
UV雕刻工具

用于操作UV的笔刷。所有笔刷使用喷枪笔触方法：只要按住 LMB 不放，笔刷会持续作用。

抓起 G
抓起笔刷用于拖动UV。
放缓姿态 S, Shift-LMB

放缓姿态笔刷用于使UV分布更加均匀。该算法基于空间，而非最小化拉伸，所以大多数时候需要使用最小化拉伸以获得理想效果。不过，在缝合孤岛后或者展开结果乱七八糟后平滑UV分布，使用该笔刷再好不过。

松弛模式

有两种松弛方法:

拉普拉斯, HC

拿捏 P
拿捏笔刷用于向笔刷中心移动UV顶点。可以按下 Ctrl-LMB 反转拿捏效果。
显示笔刷
显示/隐藏雕刻游标。
