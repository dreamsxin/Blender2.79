进阶选择¶

编辑模式下的选择菜单还有一些其他的选择工具：

锐边
此工具选择两个面夹角大于设定角度值的交线。角度值越大，选中的边越锐利。
相连的平展面 Ctrl-Shift-Alt-F
基于面之间的夹角阈值，选择与之相连的面。这在选择平坦面时很管用。
镜像
选择指定坐标轴镜像方向上的对应网格元素。
活动项的同侧
选择在某一轴向上，位于活动顶点同一侧的所有网格中的顶点。仅适用于顶点选择模式。
关联项
Selects all components that are connected to the current selection. (see Select Linked).

间隔式弃选¶

参考

模式: 编辑模式
菜单: 选择 ‣ 间隔式弃选

该工具对已选元素应用交替选择模式，仅适用于已选网格元素多余一个的情形。

按指定间隔选择
使用当前选中元素，每隔指定数量元素弃选一次。
跳过
弃选时跳过的步数。
偏移量
开始点的偏移量。

按特征选择¶

非流形点/线 Ctrl-Shift-Alt-M

选择一个网格的 非流形 几何体。这个选项在编辑网格的时候有用，只用在点和边的选择模式下。这个操作面板提供几个选择选项：

扩展选择
在当前选中项基础上加选。
线框
选择所有不属于任何面的边。
边界
选择边界和洞上的边。
多面
选择属于三个及以上面的边。
非连续性的
选择属于两个法线方向相反的面的共用边。
顶点
选择属于 线框 和 多面 边上的顶点，独立顶点，还有属于非相邻面的顶点。
内侧面
选择被两个以上的面共用的所有边线。
按侧选面
选择包含指定边数的面。
松散几何元素
选择没有形成面的所有顶点和边。
未归组顶点
选择所有不属于 顶点组 的顶点。

选择关联项¶

参考

模式: 编辑模式
菜单: 选择 ‣ 关联项
快捷键: Ctrl-L

选择和现有的元素相互联系的几何体。当一个网格有断开，重叠的部分，但是想将它独立出来的时候非常有用，其他方式就很单调乏味。

为了更多的控制操作，你可以使用操作面板的划界，因此选择受制于缝合边，锐边，材质或者UV。

使用 拾取关联 你可以直接选取游标下相联系的几何体，使用 L 捷径选择或者 Shift-L 去取消选择关联项。

这以不同的方式工作，因为它使用光标下的几何体而不是现有的选择。

选择相似¶

参考

模式: 编辑模式
菜单: 选择 ‣ 相似
快捷键: Shift-G

基于激活工具后可以在工具属性中设置的选项，选择和已被选择的几何体有相同的确定的属性的几何体。工具选项根据选择模式而改变：

点选择模式：
法线
选择所有和现在当前被选择的点法线方向相同的点。
相邻面数量
选择所有和它们有着同样数量相关联面的所有顶点。
顶点组
选择 顶点组 的所有顶点。
连接边数量
选择所有和它们有着同样数量相关联边的所有顶点。
面区域
基于拓扑结构选择匹配网格特性的多个相似区域。
边选择模式：
长度
选择所有和被选择的边有着相同长度的边。
方向
选择所有和被选择的边有着相同方向（角度）的边。
一条边周围面的数量
选择属于相同数量面的所有边。
面的顶角
选择两个面之间组成相同角度的所有边，就和已经选中的边一样。
折痕
选择所有与选中项拥有相似 折痕 值的边。
倒角
选择所有和现被选中的边有同样的 倒角权重 的边。
缝合边
选择所有和现被选中边有着同样 缝合边 状态的边。缝合边 是一个真/假的设定，用在 UV贴图.
锐度
选择所有和现被选中边有着同样 锐利 状态的边。 锐利 是一个真/假的设定（一种标志），用在 拆边修改器。
面选择模式：
材质
选取所有和已经选择的面有相同材质的所有面。
图像
选择所有和已经选中的面使用同样的UV贴图的面 (参考 UV贴图 页面).
区域
选取所有和已经选择的面有相同区域的所有面。
多边形面数
选择所有有着相同数量边的面。
周长
选择所有有着相同周长的面(增加边的长度值)。
法线
选择和已经被选中的面有着相似法线的所有面。这是一种选择有着相同方向（角度）的面的方法。
共面
选择所有和被选择的面基本在同一个平面的面。

增多/减少¶

增多 Ctrl-NumpadPlus
拓展选择类型中相邻元素的选择范围。
减少 Ctrl-NumpadMinus
缩小选择类型中相邻元素的选择范围。
下一个活动范围 Ctrl-Shift-NumpadPlus
这个选项过去是用来选择基于附近拓扑结构的下一个点/边/面。
上一个活动范围 Ctrl-Shift-NumpadMinus
选择移除最近一次选择的元素后的此前的元素。

选择循环¶

你可以轻松地选择循环的元素：

循环边¶

参考

模式: 编辑模式 --> 点或者边的选择模式
菜单: 选择 ‣边循环
快捷键: Alt-RMB

当选择一条边的时候按住 Alt 选择在一条线从头到尾相联系的循环边，在鼠标下穿过这条边。按住 Alt-Shift 时候添加当前的选择。

循环边的选择基于当前被选择的边，使用 Select ‣ Edge Loop，或者在 边专有项 菜单 边循环选择 选项中 Ctrl-E.

Note

顶点 模式

在 顶点 选择模式下，你可以选择循环边，通过使用相同的快捷键，并且点击 边 (不是顶点).

纵向和横向的循环边。

左边的圆球展示了纵向选择的一条边。注意这个环不是闭合的。这是因为算法在极点处选中顶点并终止，这是因为极点上的顶点连接到超过4条边。

循环面¶

参考

模式: 编辑模式 --> 面或者点的选择模式
快捷键: Alt-RMB

在面的选择模式下，当选择一条 边 时按住 Alt 去选择一个这条边上相联系的从头到尾的面循环，穿过和这条边垂直的方向。

在点选择模式下，可以同样的使用 Ctrl-Alt 完成选择一条边的任务，它隐式地选择面循环。

面循环选择。

这个面循环是在一条边上按住 Alt-RMB 被选中的，在 面 选择模式下。这个循环从被选中的边垂直延伸。

Alt 与 Ctrl-Alt 在点选择模式下.

一个面循环可以在 点 选择模式被选中。从技术上来讲 Ctrl-Alt-RMB 会选择 并排边 ,然而，在 点*选择模式下，选择一个并排边 含蓄地选择了一个 面循环 ，这是当选择了面对应的另外对应的边隐式地选择了整个面。

边缘边界¶

参考

模式: 编辑模式 --> 点或者边的选择模式
快捷键: Alt-RMB

边缘边界下的循环选择。为了拓展所有边界的选择并且当前边界已经选中再一次使用 Alt-RMB .

并排边¶

参考

模式: 编辑模式
菜单: 选择 ‣并排边
快捷键: Ctrl-Alt-RMB

在 边 选择模式下，当选中一条边(或者两个顶点时)保持选择一系列没有相互联系的边按下 Ctrl-Alt ，但是彼此相反的两侧依然保持 face loop.

与边循环一样，你也可以基于当前选择使用 选择 ‣ 并排边, 或者在 边专有项 菜单 Ctrl-E 中使用 并排边。

Note

顶点 模式

在 点 选择模式下，你可以在 点击边 (而不是顶点）时使用相同的快捷键，但这将直接选择相应的面循环。

选定的边循环和选定的并排边。

看图. 选定的边循环和选定的并排边。 点击同样的边，但是两组不同的 "边组" 被选中是基于不同的工具。一个是计算过程中基于边的，另一个是基于面的。

最短路径¶

参考

模式: 编辑模式
菜单: 选择 ‣最短路径
快捷键: Ctrl-RMB

选择一个面或者点的路径 Ctrl-RMB.

从已经选中的顶点/边/面开始的最短路径中的几何体都会被选择。

面步进
支持点和面的对角路径，并且能选择并排边。
拓扑距离
这个是用路径边的数量计算距离而不是路径边的长度,
填充区域 Ctrl-Shift-RMB
选择最短路径中已经选中的地方到点击处的区域。
间隔式弃选
参考 间隔式弃选.

循环线内侧区域¶

参考

模式: 编辑模式 --> 边选择模式
菜单: 选择 ‣ 选择循环线内侧区域

选择循环线内侧区域 是选择一个闭合环里的内部区域。虽然可以在 点 和 面 选择模式中使用这个操作,但是结果可能无法预料。应该注意的是如果所选的环并不是闭合的，那么网格中所有相关联的边都会被认为在闭合环区域里。

循环区域。

如你所见，这个工具可以处理多个循环区域。

这个工具处理“洞”也是很棒的。

区域轮廓线¶

参考

模式: 编辑模式 --> 边选择模式
菜单: 选择 ‣ 选择区域轮廓线

选择区域轮廓线 是和 选择循环线内侧区域 相对应的，基于当前所选的所有区域，它只选择这些区域的边界（轮廓）。它可以在任何选择模式下操作，但是 面 模式下操作会转成 边 选择模式。
