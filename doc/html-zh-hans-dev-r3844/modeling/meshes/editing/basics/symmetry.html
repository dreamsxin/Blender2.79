对称¶

吸附到对称结构¶

参考

模式: 编辑模式
菜单: 网格 ‣ 吸附到对称结构

该工具用于将网格顶点吸附至其镜像位置。

用于处理整体对称的网格，但是存在被移动过的顶点，导致Blender无法侦测其为对称(比如开启X向镜像选项时)。

这可能是在未开启X向镜像时无意导致的。有时候，从其他软件导入的模型，在不够对称时，也会导致镜像失败。

方向
指定吸附的轴和方向。可以选择三个轴向之一，正向到负向，或负向到正向。
阈值
指定查找配对顶点时的搜索半径。
系数
支持从镜像位置到目标位置的混合(0.5表示等比例混合)。
中心
吸附中心轴附近的顶点至轴心。

吸附到对称结构后。

对称¶

参考

模式: 编辑模式
菜单: 网格 ‣ 对称

对称 工具是一种使网格快速对称化的方法。 对称 通过在物体的轴心点切开物体，然后沿指定轴向对几何体进行镜像，然后合并两半(如果是相连的)。同时其他网格数据(比如UV，顶点色、顶点权重)也从一侧复制到另一侧。

方向
指定作用的轴和方向。可以选择三个轴向之一，正向到负向，或负向到正向。
阈值
ToDo.

对称前网格。

对称后网格。

See also

关于镜像的更多信息请访问 镜像 页面，该工具用于沿对称轴翻转几何体。
