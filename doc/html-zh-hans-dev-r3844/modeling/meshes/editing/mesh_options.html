网格选项¶

参考

模式: 编辑模式
面板: 工具栏 ‣ 选项 ‣ 网格选项面板

X向镜像¶

网格选项 中的 X向镜像 选项用于通过仅编辑一侧顶点，来完成成对的对称顶点编辑操作。对一个元素进行变换(顶点、边或面)，如果存在确切的X向镜像对应元素(自身空间)，会对对称部分应用沿其自身X轴向对称的变换。

Note

X向镜像 的使用条件相当严格，所以很难使用。要获得(半)网格的准确镜像，镜像修改器 来得更加简单和方便

拓扑镜像¶

Note

要使用 拓扑镜像 ，必须先勾选 X向镜像。

在使用 X向镜像 选项时，网格几何体的顶点必须完美配对。如果其位置无法准确配对， X向镜像 不会将其当做镜像顶点处理。

拓扑镜像 试图通过综合考虑顶点位置以及与几何体中其他顶点的相对关系，来确定哪些是镜像顶点。该工具通过审视整体拓扑结构，来确定是否将具体顶点认定为镜像点。在同时勾选 X向镜像 和 拓扑镜像 之后，非对称顶点也可以作为镜像顶点，并当做镜像处理。

Note

几何体越复杂， 拓扑镜像 功能更加可靠。如果处理的是非常简单的几何体，如 立方体 或者 经纬球 ， 拓扑镜像 选项通常是不起作用的。

范例¶

举个例子，说说如何使用 拓扑镜像，打开一个新的Blender场景，然后删除3D窗口中默认的立方体，再添加一个猴头。

  1. 按下 Tab 键，切换猴头进入 编辑模式。
  2. 取消勾选 X向镜像，稍稍移动猴头的几个顶点。
  3. Then Turn X Mirror option on again but leave Topology Mirror disabled.
  4. 现在再次移动顶点， X向镜像 不起作用，镜像的顶点不会随之修改。
  5. 如果接着勾选 拓扑镜像 ，再次移动相同顶点， X向镜像 会对随之操作镜像顶点，即使这些顶点不是完美对应的。

更多选项¶

边选择模式

使用 Ctrl-RMB 选择边时，对选中边执行的操作。

选择
仅选中边。
缝合边
标记边线为UV展开的缝合边。
锐边
标记边线为用于拆边修改器的锐边。
折痕
标记边线为用于表面细分修改器的折痕，权重为1.0。
倒角
标记边线的倒角权重为1.0(用于倒角修改器)。
实时展开
勾选 实时展开，每次修改边线的缝合属性时，自动重新解算UV展开。
重合点阈值
指定 自动合并编辑 工具使用的最大顶点距离。
