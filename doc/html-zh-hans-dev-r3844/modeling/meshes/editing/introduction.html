简介¶

Blender提供了各种网格编辑工具，用于添加、 复制、 移动和删除网格元素。

这些工具可以通过 网格工具 面板，3D视图标题栏上的网格菜单，3D视图中的上下文菜单访问，还有一些可以通过独立的快捷键调用。

Note

"变换精度/吸附" 快捷键 Ctrl 和 Shift 同样适用于这些高级操作... 然而，它们之中的大部分都没有 轴向锁定 功能，并且还有一些操作不考虑 轴心的 和 变换坐标系。

这些变换工具可以在标题栏的 网格 菜单中的 变换 子菜单中找到。注意其中一些工具还可以用在其他可编辑的物体上，如曲线，曲面和晶格。

访问网格工具¶

这些网格工具可以在多个位置找到，还可以通过快捷键访问。

网格工具栏¶

选中一个网格， Tab 进入编辑模式，并且 工具栏 由 物体工具 变成 网格工具。但这只是一部分网格编辑工具。

菜单¶

网格 菜单位于标题栏。有一些菜单可以通过快捷键访问: Ctrl-F 调出面工具菜单； Ctrl-E 调出边工具菜单； Ctrl-V 调出点工具菜单
