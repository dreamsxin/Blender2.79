基本体¶

参考

模式: 物体模式 和 编辑模式
面板: 工具架 ‣ 创建 ‣ 添加基本体/曲线
菜单: 添加 ‣ 曲线
快捷键: Shift-A

添加曲线菜单。

在物体模式下，添加曲线 菜单提供了5种曲线基本形式:

Bézier Curve 贝塞尔曲线¶

添加一段由2个控制点构成的2D贝塞尔曲线。

Bézier Circle 贝塞尔圆环¶

添加一个闭合的2D贝塞尔圆环(由四个控制点构成)。

NURBS 曲线¶

添加一段由4个 均匀 控制结点构成的开放2D NURBS 曲线。

NURBS 圆环¶

添加一个2D NURBS 闭合圆环 (由8个控制点构成).

Path 路径曲线¶

添加一段由5个对齐的 端点 结点构成的开放3D NURBS 曲线，并启用 曲线路径设置。

Draw Curve 绘制曲线¶

使用 绘制曲线工具 拖动鼠标手绘曲线。

通用选项¶

半径，对齐到屏幕，位移，旋转
