选项¶

参考

模式: 雕刻模式
面板: 工具栏 ‣ 选项

叠加面板¶

启用时，笔刷纹理显示在视图中。

View 视图
眼睛图标是用于切换来显示或隐藏给定画笔纹理。
Alpha
你可以使用alpha滑块更改显示纹理时使用的透明度。
笔触叠加
画笔图标允许你在笔触期间关闭视图叠加层。

选项面板¶

Gravity 重力
Factor 系数
设置系数可以让您将重力添加到画笔笔触中，使其具有悬垂效果。
Orientation 参照坐标系
使用另一个物体，重力可以定向到设定物体的局部Z轴，改变重力的方向。
Threaded Sculpt 多线程雕刻
利用多个CPU处理器进程来提高雕刻性能。
Fast Navigation 快速导航
对于 multiresolution多级精度 模型，在视口中导航时显示低分辨率。
Use Deform Only 仅使用形变
在激活对象上限制形变修改器，多级精度修改器。建构性修改器（如细分曲面，镜像等）被取消激活，因为它们可能会给出不准确的结果。
Show Diffuse Color 显示漫反射颜色
允许激活对象在雕刻时显示其漫反射颜色。
Unified Settings 统一设置
Size 尺寸
强制在笔刷之间共享笔刷大小。
Strength 强度
强制在笔刷之间共享笔刷强度。
Color 颜色
不用于雕刻模式。
Show Brush 显示笔刷
显示视口中的笔刷形状。
Color (Add/Subtract) 颜色（加/减)
当其特定效果激活时，设置笔刷环的颜色。

外观面板¶

Show Brush 显示笔刷
显示视口中的笔刷形状。
Color (Add/Subtract) 颜色（加/减)
当其特定效果激活时，设置笔刷环的颜色。
Custom Icon 自定义图标
将图像文件附加到激活的笔刷上作为图标。
