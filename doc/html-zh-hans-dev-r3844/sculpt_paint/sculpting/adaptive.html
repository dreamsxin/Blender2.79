自适应雕刻¶

动态拓扑¶

参考

模式: 雕刻模式
面板: 工具栏 ‣ 工具 ‣ Dyntopo
热键: Ctrl-D

动态拓扑（也称为dyntopo(动态拓扑)）是一种动态镶嵌的雕刻方法，可以在操作中添加和删除细节，而常规雕刻只影响网格的形状。

这使得可以用简单的网格来雕刻复杂的形状，而不是仅仅将细节添加到建模的基本网格上。

Dyntopo可以用标题中的复选框来切换，或者使用 Ctrl-D 。在动态拓扑激活的情况下，大多数笔刷会在笔迹中细分网格。

Detail Type 细节类型

Dyntopo使用三种不同的细节方法为对象创建动态细节。

Relative Detail 相对细节
该方法使用基于像素数的细节大小，并且依次将创建该尺寸的拓扑。缩小大细节，放大小细节。
Constant Detail 常数细节
为了保持整个对象的细节统一，可以使用常量细节。细节基于单个 BU 的百分比。
Brush Detail 笔刷细节
通过这种方法，你可以根据笔刷大小创建拓扑，更好地控制拓扑。你可以通过简单地调整笔刷本身的大小来增加和降低拓扑。细节尺寸基于笔刷本身的大小，其中100％将创建笔刷环本身的大小的拓扑。
Detail Size 细节尺寸 Shift-D

每个细节类型的细节都在这里设置。根据使用的详细信息类型，此属性将显示为像素数（px）或百分比。

Sample Detail Size (pipette icon)
使用“常规细节尺寸”，可以通过单击旁边的吸管图标详细设置，然后单击该区域来对某个网格区域的细节值进行采样。
Detail Refine Method 细节细化方法

当使用动态拓扑时，将使用某种方法来说明如何处理拓扑。设置选项将决定在更改拓扑结构时将使用哪些方法。

Subdivide 细分
就像细分工具一样，这种方法只会细分拓扑，以符合给定的细节。
Collapse 折叠
当拓扑太密集且小于给定的细节时，边缘将被折叠以适合细节尺寸。
Subdivide Collapse 细分折叠
该方法结合了这两种方法，细分了细节尺寸的边缘和折叠拓扑。
Detail Flood Fill
当使用常数细节模式时，此选项可用，允许您根据详细尺寸，使用统一的详细信息填充整个对象。
Smooth Shading 平滑着色
切换网格面是平滑还是平坦的着色。在动态拓扑模式下，所有面都具有相同类型的着色。
Optimize 优化
如果在启用动态拓扑时雕刻开始变慢，请使用 优化 按钮重新计算雕刻BVH。
Direction 方向
确定模型对称的方向。
Symmetrize 对称
使用方向定向对称。由于Dyntopo添加细节动态可能会发生模型不对称，所以这是一个很好的工具。

Multi-Resolution Modifier 多级精度修改器¶

需要多级精度修改器才能雕刻。这个修改器将细分网格。更多的细分需要更多的计算。使用Blender堆栈不破坏数据，当你想有一个干净的拓扑基础网格时，多级精度修改器将有所帮助。

当使用多级精度修改器雕刻时，我们可以在不同级别的细分中进行雕刻，这意味着我们可以在细分级别1中雕刻一些细节，并在细分2中添加更多细节，并返回到细分1纠正一些错误。虽然经常使用此工作流程，但多级精度修改器仍有一些限制。你可能会遇到一些网格失真问题。作为一个建议，添加更多细节之前添加更多的细分。粘土笔刷，绘画笔刷通过多级精度修改器雕刻更好地雕刻二次形式。

  • 加强一个细分级别 PageUp
  • 降低一个细分级别 PageDown
  • 设置细分级别 Ctrl-0 至 Ctrl-5
