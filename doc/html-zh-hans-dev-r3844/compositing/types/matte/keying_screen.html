屏幕抠像节点¶

屏幕抠像节点。

屏幕抠像 节点创建了关键帧集节点作为颜色参考使用的底片。在影片剪辑中的运动追踪点进行颜色采样生成渐变色。它可用来处理在绿屏背景上不平整的颜色。

输入¶

此节点无输入连接端口。

属性¶

影片剪辑
选择的剪辑数据块被用于渐变色的输入值。
追踪对象
追踪对象生成为渐变图。用户可以在 Object panel，创建新的追踪对象，因为追踪到的路径渐变图并不能应用于 摄影机/对象 追踪。追踪进行之后的渐变图会将其放置在可以被采样的地方。这些追踪路径也可以手动进行移动，因此渐变图会随着影片剪辑进行更新。追踪路径可以方便地进行常规的屏幕偏移。

输出¶

滤色
渐变图像输出。

如果增加关键帧集节点的容差范围，将在绿屏背景上产生更多的阴暗处。对前景的对象遮住过多也是不正确的。

取而代之的方法是增加可接受阴暗部分的范围使其被遮挡， 屏幕抠像 节点让用户可以改变绿屏背景的阴暗部分（或者其他颜色背景）以使用图像的不同部分。

从 Movie Clip Editor。开始。打开属性区和工具架显示出追踪布局。生成的渐变路径是不能用于摄影机解算，因为追踪路径的结果不理想。因此在 物体 选项卡创建一个新的追踪对象，将追踪标记放置在影片剪辑中背景的不同部分。

这些追踪路径也可以手动移动位置，渐变图也随着时间帧进行更新。如果追踪标记在某一帧不可用，就不能用来创建渐变图。（比如在如图所示的手臂上出现的红色标记）

一旦追踪被创建，在合成配置中添加节点，为背景图选择追踪对象。

带有 屏幕抠像 节点配置生成渐变底片，连接到关键帧集节点的颜色输入端口上。
