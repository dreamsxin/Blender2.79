遮片类节点¶

此类节点是合成中的基本工具节点之一，可以给没有 Alpha通道 的图像创建一个 遮片 。在蓝屏或绿屏布素材中，前景活动的画面可以通过绘制一个遮片形状来和蓝色，绿色背景进行分离。

通常，这类节点可以连接到预览器节点，在合成窗口左下角的 UV/图像 编辑器中可以显示预览器节点的输出结果，在素材的样本图像中可以使用滑块进行实时的播放，右侧也包含了其设置信息。在一些情况下，轻微的调节就可以去除图像上的瑕疵。过分的除去绿色信息会导致前景的演员角色看起来发青或者是带有紫色。

用户可以将这些节点连接起来，对遮片和颜色进行校正形成更好的结果。每个节点会在上游节点设置结果的基础上进行增强。 抠像节点 对于绿屏背景抠像接近于全能型节点，但好的合成结果是由多个节点进行组合才能做到。

Note

垃圾遮罩不是一个具体的节点，它是将图像上不需要的部分去除掉的一种技巧。 遮罩 用于将识别出的内容从图像上去除掉，不好处理的部分可以通过像色度抠像的方法进行移除。我们可以选择特定的内容进行移除，或者是反转选择起初的操作素材，或者是移除其他不需要的部分。

一些节点自身可以实现垃圾遮片处理的效果，如果某个节点的添加实现不了预期效果，用户还可以将其从垃圾遮片节点树中移除此节点。

简单的垃圾遮片可以通过 方形遮罩 或者 椭圆遮罩 创建，更加复杂的垃圾遮片形状可以使用 双边遮罩 或者 遮罩。
