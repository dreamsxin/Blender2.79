合并/分离节点¶

这类节点本质上都是做同一件事情：

  • 分离：将一幅图像的颜色通道分离成单独颜色通道。
  • 合成：将单独的颜色通道 重新/合成 为单一的混合颜色通道。

These nodes can be used to manipulate each color channel independently. Each type is differentiated in the applied color space.

在合成和相关的贴图中都支持Alpha通道。在贴图的中只允许使用RGB色彩空间。在Blender内部着色器节点的HSV或Cycles 着色器都提供了额外的可以 合并/分离 矢量（XYZ）的节点。

合成节点也可以用于输入单独的颜色值。在RGBA和HSVA颜色空间推荐使用 RGB节点 。一些简单的操作可以通过 Color Nodes 来实现。

Separate/Combine RGBA Nodes¶

合并RGBA节点。

分离RGBA节点。

输入/输出¶

图像
标准图像的输入/输出。
  • R (红)
  • G (绿)
  • B (蓝)
  • A (Alpha通道)

属性¶

这个节点没有属性。

范例¶

第一个范例中，首先模糊Alpha通道，然后和其他颜色进行合并。当放置在场景中，通道的边缘会发生颜色融合，而不是一个硬朗的边缘。这个可以称作三维场景中的抗锯齿功能。利用这个节点设置，可以添加有动作的CG元素并移除硬表面的硬边缘。运动物体边缘缩放会产生物体的相位进出，称作“脱相”时间旅行同步特效。

在这个节点设置中，所有的红色变为绿色，所有的绿色变为红色和蓝色，或从图片中把蓝色完全移除。

图像
标准图像的输入/输出。
  • Y（亮度）
  • U (U 色度分量)
  • V (V 色度分量)
  • A (Alpha通道)

分离/合并 YCbCrA节点¶

合并 YCbCrA Node。

分离 YCbCrA节点。

输入/输出¶

图像
标准图像的输入/输出。
  • Y（亮度）
  • Cb (色度 蓝)
  • Cr (色度 红)
  • A (Alpha通道)

属性¶

模式
不同的色域，应对不同的投放环境。ITU 601（SDTV色彩空间，标清广播色域）, ITU 709（HDTV色彩空间 高清广播色域）, Jpeg（图像压缩标准）

Tip

若在颜色渐变节点上调节这些颜色通道的数值，差值类型应选择“原始”方式计算会得到较好的呈现结果。使用“指数”缩放亮度通道产生高对比度效果。
