界面¶

界面配置用于修改UI元素显示和回馈的方式。

显示¶

工具提示
启用后，将鼠标指针在控件上方悬停，就会显示工具提示，解释鼠标指针下方控件的作用，对应的快捷键（如果有快捷键的话)，以及其调用的Python函数。
Python 工具提示
在工具提示下方显示对象的Python信息。
物体信息
在3D视窗的左下角显示活动物体名称和当前帧数。
大号光标
勾选后使用大号光标。
视图名称
在3D视窗的左上角显示当前视窗的名称和类型。比如： 用户视图(透视) 或者 顶视图(正交)。
播放帧速
播放动画时，在窗口左上方显示每秒刷新帧数（fps），如果无法到达指定帧速，数值显示为红色。
全局场景
强制所有屏幕显示当前场景(一个工程文件可以包含多个场景)。
物体原点尺寸
视图中3D物体中心直径(可调整范围为4至10像素)。
显示小轴
在视窗的左下角显示坐标系的小轴。
尺寸
小轴的尺寸。
亮度
调整小轴的亮度。

警告¶

退出时提示
退出Blender时，弹出对话框询问是否真想退出(目前仅适用于MS-Windows)

视图控制¶

游标视深
定位游标时使用鼠标下方的深度。
自动深度
使用鼠标的深度值来改善视图的平移、旋转、缩放功能。可结合 放大至鼠标的位置 使用。
放大至鼠标的位置
启用后，鼠标指针的位置将会替代2D视窗中心成为缩放的焦点。这会在频繁内外翻转缩放时有助于避免发生位移。可以避免频繁缩放视图时的平移操作。
围绕选择物体旋转

选中物体(边界盒中心)成为视图的旋转中心。当无选中物体时，使用上一次操作选中的物体。

Hint

这似乎是理想的方法。然而，在大型物体如地形网格时，会出现问题，因为其中心未必是你的关注点。

全局转轴
将所有3D视窗的旋转/缩放轴心锁定为同一点。
摄像机父级锁定
当相机被锁定且处于飞行模式时，变换其父级物体而非相机自身。
自动透视模式
顶视/侧视/前视图自动切换正交视图，其余视图自动切换透视视图。禁用后，调整视角时维持原来的正交或透视视图(切换视角前的视图类型)。
平滑视图
通过小键盘切换视图视角(顶/侧/前/相机...)时的动画时间。数值调为0，取消动画。
旋转角度
使用 Numpad4, Numpad6, Numpad8, 或者 Numpad2 旋转3D试图时的旋转角度步长值。

2D 视口:¶

最小栅格间距
2D视窗（比如顶视图/正交）中栅格线之间的最小像素间距。
时间码样式
时间码是不使用帧为时间单位时（无时间线等显示帧的窗口时）显示时间的格式，此格式用“+”来分隔次级帧数，必要时使用左右两侧的分隔线。
Zoom To Frame Type

How zooming to frame focuses around current frame.

操纵件
启用和关闭操纵件。
尺寸
操纵件的尺寸。
控制柄大小
操纵件控制柄大小，用操纵件半径(尺寸/2)的百分比表示。
热区
操纵件手柄的单击响应范围大小(像素)。

菜单:¶

鼠标划过时开启菜单
勾选此选项之后，把鼠标指针悬浮在菜单按钮上就能开启菜单，这样就不用点击啦~
菜单开启延迟
开启菜单的时间。
主菜单
菜单打开前会有1/10秒的延迟(需启用 鼠标滑过开启菜单)。
次级菜单
与上面一样，仅作用于次级菜单（例如: 文件 ‣ 打开近期文件）

菜单盘¶

Animation Timeout
打开饼菜单时的动画长度
Recenter Timeout
窗口系统会尽量使饼菜单位于窗口边界内。饼菜单会在这段时间内(1/100秒)使用鼠标初始位置作为中心。这样可以使用快速拖动选择。
半径
饼菜单选项到中心的距离(像素)。
阈值
执行菜单选择前到中心的距离。
Confirm Threshold
执行菜单选择后的距离阈值(0为禁用)。
