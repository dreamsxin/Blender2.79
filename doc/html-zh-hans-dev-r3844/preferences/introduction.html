简介¶

本章解释了如何使用 用户偏好设置 编辑器更改Blender的可自定制设置。

Blender的 用户设置 编辑器中包含用于控制Blender的运作模式的设置。

打开用户设置¶

通过 文件 ‣ 用户设置 菜单打开 用户设置 编辑器。

配置¶

打开用户偏好设置编辑器，可以自定义Blender为个人喜好的样子。在编辑器的顶部，有7组标签页:

界面
更改UI元素显示和反馈的方式。
编辑
控制多个不同的工具同时与你输入的值互动。
输入
通过规定你的按键表单来定制Blender的按键操作。
插件
通过内置新的程序片段来获得Blender软件中默认未开启的功能。
主题
定制用户界面的外观和颜色。
文件
配置自动保存设置以及给Blend文件和渲染图像设置默认文件路径，等等。
系统
设置分辨率，编写控制台设置，调整声卡和显卡设置，以及更改系统语言。

保存用户设置¶

自定义设置后，必须手动保存，不然这些设置就在下一次重启之后丢失。

在 用户设置 编辑中，点击底部左侧的 保存用户设置 按钮，可以直接保存新的用户设置。

Hint

最好对用户设置进行备份，以防丢失用户设置。

用户设置存储路径参考 目录布局 章节。

加载初始设置¶

用于还原默认配置，要注意的是，这并非永久修改，除非保存用户设置。

打开 文件 ‣ 恢复初始设置 ，然后还可以通过 用户设置 编辑器保存设置。
