输入¶

在输入设置中，可以自定义Blender与鼠标和键盘之间的交互，同时自定义键位映射。

交互¶

交互预设
可以根据个人喜好，设置和选择预设，实现与其他软件相似的使用体验。

鼠标¶

模拟 3 键鼠标

可以通过配置Blender，与其他指针设备(比如双键鼠标，苹果的单键鼠标或者笔记本的触控板)一起使用。可以使用下表中的键盘/鼠标组合，模拟三键鼠标的功能。

支持的鼠标硬件的对应快捷键¶
三键鼠标 双键鼠标 苹果鼠标
LMB LMB LMB (鼠标按钮)
MMB Alt-LMB Alt-LMB (Option/Alt 键 + 鼠标按键)
RMB RMB Cmd-LMB (Command/Apple 键 + 鼠标按键)

本手册中鼠标/键盘的组合可以替换为下表中的组合。例如:

  • MMB 拖动对应 Alt-LMB 拖动。
  • 对于单键鼠标，Shift-Alt-RMB 对应 Shift-Alt-Cmd-LMB。
持续拖拽

这个功能是用来解决拖拽或平移视图等操作被屏幕边界限制的问题。

这是通过将鼠标指针重新定位到视窗实现的。

Note

持续拖拽仅支持 相对 定位输入装置(比如说鼠标，追踪球，追踪板)。

不过，由于数位板通常使用的是 绝对 定位，所以使用数位板时该功能失效。

Blender会侦测每个动作，所以使用数位板并不会影响鼠标指针的 持续拖拽 输入效果。

拖动阈值
一个用户界面项目被认定为移动所需要的移动像素的值。阀值调节触发调节事件所需移动的像素值。
选择键
你可以选择使用鼠标左键或右键来选择物件（剩下的另一个鼠标按键是用来放置3D游标的）。
双击
双击的间隔时间(毫秒)。

Note

模拟鼠标的功能只有当 选择键 设置为 右 的时候才生效。

模拟数字键盘¶

数字键在Blender中使用相当频繁，这里的数字键指的不是常规的数字键。如果用的键盘没有数字键区(如笔记本)，可以让Blender将标准数字键视为小键盘数字键。只需勾选 模拟数字键盘。

视角操控¶

旋转方式

选择按住 MMB 旋转3D视图的交互方式。

转盘

旋转视图保持地平线水平。

这就像一个陶工轮或唱机，可以进行两个轴向的旋转，整个世界对在其中的 上 和 下 有很好的定义。

缺点使用 转盘 风格是物体操作时会失去一些灵活性。但是，你获得“向上”和“向下”的概念，如果你感觉无所适从它可以帮助你。

轨迹球
限制较少，允许任何方向。
缩放风格

选择偏好的 Ctrl-MMB 缩放风格

比例
比例 缩放取决于你按下鼠标按键时光标的位置。按住Ctrl-鼠标中键从屏幕边缘向中心拖拽进行视图缩小。按住 Ctrl-MMB 从视窗中心向边缘拖动鼠标进行视图放大。
持续
尺寸 缩放选项允许通过移动鼠标，调整到 Ctrl-MMB 的鼠标起始点的距离，控制缩放速度(而非比例)。向起始点右侧或下方移动鼠标，进行视图缩小。向起始点左侧或上方移动鼠标，进行视图放大。移的越远，缩放速度越快。通过 垂直 和 水平 单选按钮以及 反转鼠标缩放方向 选项切换缩放方向。
滑动
滑动 缩放和 持续 缩放类似，除了滑动缩放的缩放速度是恒定。
缩放轴向

用于 MMB 缩放的轴向。

垂直
向上拖动为放大，向下拖动为缩小。
水平
向左拖动为放大，向右拖动为缩小。
反转鼠标缩放方向
反转 滑动 缩放和 持续 缩放的缩放方向。
反转滚轮缩放
反转鼠标滚轮缩放的缩放方向。

浏览模式

键位映射编辑器¶

键位映射编辑器是用于修改默认快捷键，可以修改任一编辑器的键位映射。

键位映射编辑器。

键位映射预设
预定义的键位映射选项。
  1. 选中需要修改的键位映射，单击白箭头，打开键位映射树。
  2. 选择用于控制该功能的输入方式。
    • 键盘：只允许快捷键和组合快捷键，如 E, Shift-E。
    • 鼠标：左/中/右单击，并可搭配 Alt, Shift, Ctrl, Cmd 键使用。
    • NDOF: ToDo.
    • 拖拽：单击并拖动，也可搭配上述四个功能键使用。
    • 文字输入: 输入文字使用该功能。
    • 计时器：用于控制基于时间区间内的动作，比如，默认情况下， 动画步骤 使用 "计时器0" 平滑视角 使用 "计时器1"。
  3. 自定义你喜欢的快捷键，只要点击快捷键输入框，输入新的快捷键就行。

如果想要恢复一组键位映射，点击右上角的 Restore 按钮即可。

Tip

可以通过为鼠标和键盘操作新增 预设 来创建键位映射，而无需删除默认键位映射。

导入/导出键配置¶

在一些情况下，你可能需要将配置信息保存到外部文件(比如，安装新系统或者将键位映射配置共享至社区)。只需点击标题栏的 导出键配置 按钮即可实现。接着，会进入文件浏览器选择配置文件存储位置。 导入键配置 按钮用于安装电脑中保存的键位映射配置。

键位映射只包含用户修改过的键位映射和分类。另外，插件可能会为各个功能注册键位映射，不过，这些键位映射不会被导出，除非用户做过修改。导出的文件可以当做是 键位映射变化 ，而非全键位映射输出。
