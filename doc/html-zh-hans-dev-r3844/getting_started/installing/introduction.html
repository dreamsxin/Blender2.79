简介¶

Blender提供Windows, Linux以及 Mac平台下的程序下载。

最低配置要求¶

请查看你的系统是否符合 最低或推荐配置要求.

一定要将显卡驱动升级到最新版，确认OpenGL得到良好支持。

关于外设的支持，比如3D鼠标和绘图板的支持请看后面的 支持硬件列表.

下载 Blender¶

Blender基金会通过三个途径发布Blender程序，你只需要选择你觉得合适的就行。

这些选择项涵盖了所有受支持平台的二进制安装包和源代码。如果是用二进制包安装，你可以选择一个稳定发布版或者每日编译版。前者的好处是更加可靠，后者则可以提供最新的仍在开发的功能特性。Blender大约每3个月发布一次版本。你可以通过 发布说明 来了解最新的变化。

最新稳定版
这是Blender最新版的二进制发布版。非常稳定和有良好兼容的。
每日编译版
这是每日更新的二进制发布版，包含了最新的开发变化。该版本与稳定版相比是未经历全面测试，并可能崩溃的，尽管这也是官方版本，且通常并非高度试验性。
从源代码编译

Blender提供源代码供参考和安装，具备以下优势：

  • Blender is always up to date.
  • It allows access to any version or branch where a feature is being developed.
  • It can be freely customized.

Note

这里提到源码编译只是为了手册的全面性，并不 奢求常规用户自己去编译Blender程序。
