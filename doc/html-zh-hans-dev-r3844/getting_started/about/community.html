Blender社区¶

即使在闭源阶段，Blender也是可以免费获取的，这很大程度上帮助Blender获得了社区的接纳。自1998年起，一个大型、稳定并且活跃的用户社区已经围绕Blender聚集起来。2002年，社区展示了其支持的力量，在7周内筹集了10万欧元，帮助Blender转变为 GNU GPL 许可证 下发行的开源软件。

独立站点¶

有 几个致力于Blender的网站 如论坛、博客、新闻和教程网站。

Blender Artists 是最大的社区论坛之一，Blender用户聚集在这里展示他们的作品，获得反馈，提问和提供帮助，并且一般都是讨论Blender。

获得支持¶

Blender社区是其最大的特性之一，所以除手册之外，还有很多从其他用户获得帮助的方式，如 IRC 和 Stack Exchange 。

还有一些更加官方的支持资源，如 认证培训师 和 Blender Cloud。如果你发现了Blender 的问题，可以 提交bug。

在 支持 可以找到更多关于支持的细节。

开发¶

由于开源，Blender大多数开发工作都是由志愿者独自完成的。开发之间的交流多数是通过以下三个平台：

  • developer.blender.org 系统
  • 各种 邮件列表
  • #blendercoders IRC频道 (见下文)

如果你对帮助开发Blender有兴趣，可以打开 参与 页面。

IRC频道¶

关于实时讨论，在Freenode网络上有一些Blender IRC频道。可以使用IRC客户端加入这些频道：

  • #blender 从社区获取答案。
  • #blenderchat 常规讨论或水群聊天。
  • #blendercoders 用于开发者讨论Blender开发。
  • #blenderpython Python API的开发支持。
  • #gameblender 游戏引擎开发游戏的相关问题讨论。
  • #blenderwiki Blender文档相关讨论。

Note

如果你还没有IRC客户端，可以使用 网页 访问IRC。

还有几个Blender相关频道没有列出(如特定语言的Blender用户频道)。我们建议你搜索Freenode查看全部频道。
