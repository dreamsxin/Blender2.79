场景属性¶

场景¶

摄像机
用于选择相机作为活动相机。还可以在3D视图使用 Ctrl-0 设置活动相机。
背景

可使用一个场景作为背景，当用于想专注于动画的前景，又不想背景元素碍事时这很有用。

该场景可以有自己的动画，物理模拟试验等，但如果用户要编辑其中的任何内容，必须从 场景 数据块菜单中选择。

可以使用一个布景作为另一个场景的背景(可以嵌套)。所以用户可以使用现有场景作为新场景的背景，并在其基础上再做补充。

Tip

还可以结合 关联到场景 一起使用，当blend文件包含该环境，可以在很多地方重复使用。

当前激活的剪辑
激活的影片剪辑，用于约束以及视口显示。

单位¶

长度预设
常用单位比例。
长度
无
使用 Blender单位.
公制, 英制
长度计量标准单位。
角度

角度计量标准单位。

角度，弧度

Tip

当使用 角度 时，工具提示会显示弧度值。

单位比例

用于 Blender单位 与 公制/英制 单位之间转换的比例尺。

Tip

通常用户会设置比例尺为 长度 预设值，这样就无需计算数值转换了。

分离单位
使用 公制 或 英制 时，显示双位单位，如 "2.285m" 会显示为 "2m 28.5cm" 。
