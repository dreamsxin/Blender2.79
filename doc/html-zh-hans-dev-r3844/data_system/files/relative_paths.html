相对路径¶

很多Blend文件引用了外部图像或关联的Blend文件。路径告诉Blender在哪里查找这些文件。如果外部文件已被移动，引用他们的Blend文件也会出错。

当用户指定其中一个外部文件时，默认选项是使用相对路径。Blender存储相对当前文件所在目录的部分路径。这在用户需要重新组织文件夹或移动文件时很有帮助。

使用相对路径，在外部关联文件跟着一起移动时，用户可将该Blend文件移到一个新位置。例如，用户可以发送一个包含blend文件的文件夹，并将引用的外部图片放在其子文件夹下。

在支持相对路径的情形下，文件浏览器会提供一个 相对路径 的复选框，输入文件路径时，使用双斜杠(//) 代替。

默认使用相对路径，但可以在用户设置的 文件 选项卡中更改。

Note

对于新建的 untitled blend文件不能输入相对路径。在关联外部文件前先进行保存。

Hint

如果需要重新定位blend关联的源文件的相对位置，可以使用 另存为 功能，勾选 更新相对路径 选项。
