曲线引导¶

曲线引导力场。

所述 曲线引导 用于迫使粒子跟随由定义的某个路径 Curve Object.。典型的情况是移动静脉内的红细胞，或使动物中的颗粒流动。您还可以使用 曲线引导 来塑造某些发丝。

Note

您还可以使用 粒子编辑模式 来定义路径。

由于您可以将曲线动画化为Softbody或任何其他通常的方式，您可以构建非常复杂的动画，同时保持很好的控制并将模拟时间保持在最低限度。

曲线引导 选项对粒子不起作用。相反，您必须将Angular Velocity （粒子系统选项卡）设置为 Spin 并保持旋转不变（即不要打开 Dynamic ）。

曲线引导 影响同一层上的所有颗粒，独立于其与曲线的距离。如果你有一个层次上的几个指南，他们的领域会相互叠加（你在物理课程中学到的方式）。但是您可以通过更改最小距离来限制其影响半径（见下文）。

Note

曲线引导不影响 Softbodies.

选项¶

用于曲线引导力场的UI。

最小距离
从曲线的距离，直到力场有效的全强度。如果您的退出 0，此参数什么都不做，因为该字段是有效的全强度到最大距离（或无限远）。 最小距离在3D视图中的曲线的端点处以圆形显示。
自由
颗粒寿命的分数，不用于曲线。
衰减力度
此设置控制最小距离和最大距离之间的向导强度。1 的退出意味着线性进展。

路径¶

颗粒在其寿命期间遵循 曲线引导 ，速度取决于其寿命和路径的长度。

增加
如果使用Additive，颗粒的速度也会根据下降来评估。
权重
使用曲线权重来影响沿曲线的粒子影响。
使用最大值
最大影响半径。由曲线对象周围的附加圆圈显示。

簇集¶

其他设置控制沿曲线的力场的形式。

簇集数量
颗粒在曲线（1）的末端聚集在一起，或者它们分开（-1）。
形状
定义颗粒聚集在一起的形式。+0.99：颗粒在曲线末端相遇。0：沿曲线的线性进展。-0.99：曲线开始时颗粒相交。
