简介¶

材质可以有各种各样的属性。它是所有这些属性的组合，定义了材质的显示方式，以及使用该材质物体的渲染效果。可以在材质标签页上对应面板设置这些属性。

记住，材质的外观受渲染方式(面, 线框, 体积和光晕)和所用渲染引擎(Blender, Cycles, 或 游戏)影响。使用Cycles引擎渲染的图像大多数属性只能使用节点系统控制。

下面列出了Blender渲染器和游戏引擎中用到的各种属性面板，并简要介绍了其的范围。具体控制与设置请参考相关页面。

预览
当前材质映射至基础物体上的效果预览。
漫射着色器
材质的基础颜色，以及色散模型菜单。
高光着色器
反射的高亮：颜色、强度以及色散模型菜单。
色彩渐变
在散射和高光着色器中基础颜色的变化方式。
着色
材质着色模型的各种特征属性。
透明
光线可穿透物体的选项设置。
镜射
(仅Blender渲染器): 材料的反射属性。
次表面散射
(仅Blender渲染器): 模拟半透明物体，光线穿透表面、在其中散射，并在其他位置射出表面。
发股
(仅Blender渲染器): 在表面覆盖头发、毛皮等时使用。
选项
对物体着色和上色的各种选项。
阴影
控制使用该材质的物体如何投射和接手阴影。
游戏设置
(仅Blender渲染器): 游戏引擎中物体实时渲染的控制设置。
