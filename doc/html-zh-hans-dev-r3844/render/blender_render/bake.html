渲染烘焙¶

烘焙，一般地讲，是为了加速后续的其他过程进行的预先计算。根据你所选择的选项不同，从头进行渲染可能将花费大量的时间。因此，Blender允许你为选择的物体提前 “烘焙” 渲染的一些部分。然后，当你点击渲染，整个场景的渲染将会更快，因为这些物体的颜色不需要重新计算。

渲染烘焙创建一个网格物体渲染的表面的 2D 位图图片。这些图片会被使用物体的 UV 坐标重新映射到物体上。烘焙是对每个独立网格完成的，而且仅当网格已经被UV展开的情况下完成的。虽然配置和运行会花费时间，但是这将节省渲染时间。如果你要渲染一个很长的动画，花费在烘焙上的时间要大大少于花费在渲染长动画每一帧上的时间。

在强光/阴影方案中使用渲染烘焙，例如 AO 或 区域灯光的柔和阴影。如果你为主物体烘焙 AO，你将无须为完全渲染启动它，节省了渲染时间。

使用 完全渲染 或 纹理 来创建一个图片纹理；烘焙过程的纹理可以用作进一步纹理绘图的一个起点。使用*法线*来使一个低分辨率的网格看起来像高分辨率的网格。为了实现这个，UV展开一个高分辨，精细雕刻的网格和烘焙它的法线。保存法线图，和*映射*（纹理设置）一个相似的展开的低分辨率网格的UV。低分辨率网格看起来像高分辨率，但是拥有更少的面/多边形。

优势

  • 可以显著减少渲染时间。
  • 纹理绘图变得更简单。
  • 减少了多边形数量。
  • 重复渲染变得更快，成倍减少时间。

劣势

  • 物体必须是UV展开的。
  • 如果阴影被烘焙了，灯光和物体互相之间不能移动。
  • 大的文件（比如，4096 x 4096）会是内存密集的，会像渲染一样慢。
  • 人 （人工）时间必须花在展开，烘焙和保存文件并应用纹理到一个通道上。

选项¶

烘焙模式¶

完全渲染¶

烘焙所有的材质，纹理，和光照，除了高光和 次表面散射。

环境光遮蔽¶

烘焙世界面板中指定的环境光遮蔽。忽略场景中所有的灯光。

环境光遮蔽。

标准化
标准化不使用材料设置。

阴影¶

烘焙阴影和光照。

法线¶

烘焙切线和相机空间法线（和很多其他的）到一个RGB图像。

法线。

正规空间。

正规空间

法线可以在不同的空间被烘焙：

相机空间
默认方法。
世界空间
在世界坐标系下的法线，依赖物体变换和变形。
物体空间
在物体坐标系下的法线，与物体的变换无关，和变形有关。
切空间
法线在切空间坐标，无关与物体的变换与变形。这是一个新的默认，而且在很多情况下是正确的选择，因此法向图也可以被用于动画物体。

对于材质，相同的空间也可以被选择，在图像纹理选项中紧挨着存在的*法向图*设置。为了正确的结果，这里的设置应该和用于烘焙的设置相匹配。

纹理¶

仅烘焙材质的颜色和纹理，没有渐变。

位移¶

类似烘焙法线图，位移图也可以使用*选定的到活动的*选项，从一个高分辨率的物体烘焙到一个展开的低分辨率物体。

位移。

标准化
规范化到距离。

在Blender内，当和细分表面和位移修改器一起使用时，在烘焙前临时增加一个重细分表面修改器到低分辨率模型是必要的。这意味着如果你在细分表面顶部使用一个位移修改器，位移将会是正确的，因为它被作为一个相对于细分的几何图形的差保存，而不是原始的基网格（这能通过细分表面变得显著扭曲）。烘焙时渲染细分级别越高，位移就会越精确。这个技术在保存位移图用于外部渲染器时也是有用的。

发射¶

烘焙发射，或一种材料的发光颜色。

Alpha¶

烘焙 Alpha 值，或材料的透明性。

反射颜色和强度¶

烘焙发射颜色或强度值。

镜面发射的颜色和强度¶

烘焙镜面发射颜色和镜面发射强度值。

完全渲染。

额外选项¶

清除
如果选定了，在烘焙渲染前清除图像到选定的背景颜色（默认是黑色）。
边缘
烘焙结果被扩展了很多像素超出了每个UV“岛”的边界到柔软的纹理。
分割
固定
狭缝四处可预见地 (0, 1, 2) (0, 2, 3)。
固定地备选
Slit quads predictably (1, 2, 3) (1, 3, 0)。
自动的
在烘焙的时候，分割的四分位给出最小的失真。
选择的到活动的

从其他物体到被烘焙到活动物体上启动信息。

距离
控制一个物体上的一个点可以离活动物体上的点多远。仅对*选定的到活动的*是必须的。一个典型的使用情景是制作一个细致的高多边形的物体，然后烘焙它的法线到一个低多边形数量的物体上。由此产生的法线图可以被用于制作低多边形物体看起来更细致。
偏差
Bias towards further away from the object (in Blender units).

Note

网格必须在渲染中是可见的

如果一个网格在普通的渲染中是不可见的，例如因为为了在轮廓渲染而禁用它或启用了顶点复制设置，它将不能被烘焙。

工作流程¶

  1. In a 3D View editor, select a mesh and enter UV/Face Select mode.
  2. Unwrap the mesh object.
  3. 在一个 UV/图片 编辑器内，创建一个新的图片或打开一个已经存在的。如果你的 3D 视图是在纹理显示模式，你现在应该看到映射到你网格的图片。确保所有的面都被选择。
  4. In the Bake panel at the bottom of the Render menu, bake your desired type of image (Full Render, etc.).
  5. 当渲染完成，Blender 将使用烘焙的图片替代这个图片。
  6. 保存图像。
  7. Apply the image to the mesh as a UV texture. For displacement and normal maps, refer to Bump and Normal Maps. For full and texture bakes, refer to Textures.
  8. 使用下面描述的过程调整图片，或者用 纹理绘制 或外部的图片编辑器润色。
