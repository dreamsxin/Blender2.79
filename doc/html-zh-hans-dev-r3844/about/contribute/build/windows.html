在MS-Windows中生成¶

将rst文件转换为漂亮的HTML页面。有两种生成方式。

文件浏览器

在 C:\blender_docs 文件夹中运行 make.bat 。

Tip

为 make 新建桌面快捷方式。

命令行提示符

  1. 或者打开命令行提示符，并使用 cd C:\blender_docs 更改工作路径代码库文件夹。

  2. 使用以下命令生成:

    make

这是你每次生成手册都要用到的命令。第一次(或重大变更后)生成过程需要几分钟，但下一次生成就只需几秒钟。

Note

如果您遇到以 TypeError: an integer is required (got type str) 结束的错误，你可能需要安装老版本的 Babel (Python 国际化库)。

要执行此操作，只需运行:

pip install sphinx "babel<2.0"

Viewing the Local Manual¶

一旦生成文档，所有的 html 文件可以 C:\blender_docs\build\html 文件夹里找到。可以尝试使用浏览器打开 \build\html\contents.html 文件，阅读手册。
