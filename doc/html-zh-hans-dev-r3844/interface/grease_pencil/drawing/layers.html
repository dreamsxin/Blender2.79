层¶

参考

模式: 所有模式
Panel: 属性区块‣ Grease Pencil Layers

蜡笔草图是以层的形式管理的，很像GIMP或者Photoshop®里面的图层。蜡笔层和Blender里面的其他层系统没有关联。

层的主要用途是有目的地分类管理草图(比如，布局(blocking)笔记，导演对布局(blocking)的注释 或者 引导线 ）。由于上述因素，同一图层里的所有笔画(不论修改前后)是受该层的颜色，透明度以及笔画粗细设置影响。

属性区块 (N ) 的 Grease Pencil Layersl 面板用于管理蜡笔图层。

蜡笔面板。

蜡笔数据¶

下列控件用于添加、删除或者调整图层在列表中的位置。

绘制源
场景
蜡笔数据从属于当前场景，除非活动物体已经有蜡笔数据(旧文件)。
物体
蜡笔数据从属于活动物体。2.73之前的插件要用到该选项。
蜡笔
选择用于存储图层的蜡笔数据块，关于该控件参考 数据块菜单 。
活动层

从属于场景或物体的图层 列表视图 。

父级(骨骼图标)
指示已为该图层设置父级。
锁 (挂锁图标)
锁定当前图层，不允许编辑。
隐藏 (眼睛图标) H
在绘制区域隐藏当前图层。
解锁颜色 (带箭头的调色盘图标)
解除 颜色 锁定，恢复被锁定颜色的编辑。
隔离 (挂锁图标) NumpadAsterix
限制只能修改活动图层。
隔离 (可见) (眼镜图标)
隔离 活动图层，并隐藏其他图层。
特殊菜单
复制层
创建当前图层副本。
显示全部 Alt-H
显示所有隐藏图层。
隐藏其他 Shift-H
隐藏所有非选中图层。
全部锁定/解锁
锁定/解锁所有图层，用于防止不必要的修改。
Merge Down
合并当前图层与下方图层。

Note

默认情况下，大多数操作仅作用于列表中高亮显示的 活动 层。

外观设置¶

这些设置用于修改活动层的外观。

不透明度
图层的透明度。
透视效果
使场景中位于物体后方的线条可见。
显示控制点
绘制组成笔画的控制点。
Tint
颜色
用于染色的颜色。
系数
图层 染色颜色 混合比例。
宽度修改

要对活动图层中所有笔画应用的宽度修改数值。(类似修改器)

应用 (手掌与蓝色曲线图标)
按下应用按钮后，应用宽度修改，并将宽度修改值归零。

动画¶

父级

选择 父级 物体 数据ID 。图层中笔画跟随父级变换。

类型

父级关系 类型。

物体、骨架、骨骼

锁定帧
锁定当前笔画所在帧。
删除帧
从活动蜡笔图层删除活动帧。

洋葱皮¶

洋葱皮或者叫鬼影，通过较淡的轨迹显示相邻帧，帮助动画制作人员。

蜡笔洋葱皮。

A: 使用自定义颜色
洋葱皮
复选框用于启用洋葱皮。
一直使用(相机图标)
启用后，在拖动时间轴和/或者播放动画时显示鬼影。
使用自定义颜色 (调色盘图标)
切换使用 前显帧数 和 后显帧数 修改鬼影帧颜色。
前显帧数/后显帧数
颜色
当前帧前/后帧笔画显示的颜色。
前显/后显 范围
显示前/后帧的最大数量。 0 仅显示上一/下一草图。 -1 不显示前/后帧。
