菜单¶

Blender使用各种不同的菜单来访问选项和工具。

标题栏菜单¶

信息编辑器菜单。

多数 标题栏 会列出一组菜单， 位置紧靠 编辑器类型 选择器。标题栏菜单用于配置编辑器与访问工具。如果有快捷键的话，会一起显示。

收起菜单¶

有些情况下，通过收起菜单来获得更多的水平空间可以带来不少便利。在标题栏 RMB 单击并启用收起菜单，就可以实现该操作。

在标题栏任意位置右击。

从收起图标访问菜单。

选择菜单¶

3D视图模式选择菜单。

选择菜单或者短选择器用于在一组选项中进行选择。可显示为文本与/或图标。选项显示为弹出菜单。被选中选项显示为活动状态。

弹出菜单¶

视图着色弹出菜单。

弹出菜单是重叠式的，通过右侧显示正倒三角的菜单，或快捷键调出。

如果内容过多，屏幕无法显示，会显示小三角形指示。在菜单上方移动鼠标可以滚动菜单选项。

例如， 视图着色 按钮会弹出可用着色选项菜单。

鼠标选择
在目标项 LMB 单击。
数字选择
可以使用数字键或者 Numpad 输入选项序号选择。例如 Numpad-1 用于选择第一项，以此类推。

可以拖动标题移动弹出菜单。

快捷键¶

  • 悬停鼠标时使用 Wheel 。
  • 方向键用于浏览。
  • 可以使用带下划线的字符执行菜单选项。
  • Number keys or numpad can be used to access menu items. (Where 1 is the first menu item, 2 the second, etc. For larger menus Alt-1 the 11th... up to Alt-0 the 20th).
  • 按下 Enter 执行选中的菜单项。
  • 按下 Esc 、将鼠标从弹出菜单移开或者 LMB 单击菜单外部任意位置关闭菜单。

特殊菜单¶

特殊弹出菜单包含一组上下文敏感操作列表。 通过灰色背景倒三角按钮 或者在多数编辑器使用 W 打开，提供上下文相关工具的快速访问方式。

饼菜单¶

饼菜单是一组放射状围绕鼠标分布的菜单。在用户设置 插件 ‣ Pie Menu 中启用饼菜单。

着色方式的饼菜单。

交互¶

饼菜单使用按键唤出，这些按键会在 插件偏好设置 列出。

不移动鼠标并松开按键，菜单会保持打开，用户将鼠标指针朝饼菜单某个选项方向移动，单击即可选中。移动鼠标指向饼菜单某个选项后松开按键，会关闭菜单并激活选中菜单项。

饼菜单中间的光碟部件显示当前饼菜单方向，同时选中项高亮显示。不管鼠标位于菜单中间的光碟部件上还是其外，饼菜单只有一个选中方向。

饼菜单选项支持使用菜单项中下划线字母按键加速，还可以使用数字键选择。

如果有子菜单存在，会显示加号图标。

见 饼菜单设置 。
