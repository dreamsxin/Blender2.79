启动画面¶

启动Blender过程中，启动画面出现在窗口中间，其中包含了帮助选项链接和最近打开的blend文件，详情见下文。

Blender启动画面。

单击启动画面外任意位置(必须位于Blender窗口)，或者按 Esc ， 可以关闭启动画面并启动新项目。启动画面将消失，并显示默认窗口布局。

在 信息编辑器 标题栏单击Blender图标或者选择 信息编辑器 ‣ 帮助 ‣ 启动画面 菜单，可以重新打开启动画面。

标题
除Blender图标和文字之外，还显示Blender版本，例如目前的版本就是 2.79 。
图像
通过图像可以识别包和版本。
日期
在右上角，你可以看到该版本Blender的编译日期。
Hash
Git Hash值。在诊断问题时，这个可以帮到维护人员。
分支
可选分支ID。
交互
与 用户设置 ‣ 输入 相同，用于设置快捷键配置。
链接
官方网页链接，与信息编辑器中 帮助 菜单相同。见 帮助菜单 。
最近
你最近打开的blend文件，可以快速方便地打开最近的项目。
恢复最后一次会话
Blender会试图从临时文件中恢复最后一次的项目。见 恢复数据 。
