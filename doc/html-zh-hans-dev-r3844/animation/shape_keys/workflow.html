工作流程¶

相对形态键¶

  1. 在物体模式下，通过 Shape Key 嵌板的 + 按钮，添加一个新的形态键。
  2. Basis 是基础（rest）形状。 Key 1，Key 2，等，是新添加的形态键。
  3. 切换到编辑模式，在 Shape Key 嵌板上选择 Key 1 。
  4. 变形网格（不用删除或添加顶点）。
  5. 选择 Key 2，网格将被变为基础形状（basis）。
  6. 变化 Key 2，类似的，可以添加新的形态键。
  7. 切换回物体模式。
  8. 设置 "Key 1", "Key 2"的 值，等，观察形态键间的变化。

在下图中，从左到右依次显示物体模式下的形态键：Basis，Key 1，Key 2，和混合形态（Key 1 1.0，Key 2 0.8）。

相对形态键的例子。

绝对形态键¶

  1. 同相对形态键一样，添加一组形态键序列。
  2. 取消 Relative 的勾选。
  3. 按 Reset Timing 按钮。
  4. 切换到物体模式。
  5. 拖拽 Evaluation Time 以查看形状怎样连续变化。

绝对形态键的工作流。

通过添加 driver 或 设置 keyframes 到 Evaluation Time， 你就可以创造出动画。
