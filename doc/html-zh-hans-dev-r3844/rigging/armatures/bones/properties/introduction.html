简介¶

参考

模式: 物体模式, 编辑模式和姿态模式
面板: 全部位于属性编辑器的 骨骼 标签页

选择骨骼时（在 编辑模式 和 姿态模式 ）,其属性显示在属性编辑器的 骨骼（Bone） 选项卡中。这显示了用于控制每个选定骨骼不同功能的面板; 面板根据你正在使用的模式而改变。

骨骼标签。

Relations 关系
在这个面板中，您可以安排骨骼到不同层，以便于操作。
Display 显示
显示面板允许你自定义骨骼的外观，用另一个现有物体的形状取代骨骼。
Deform 形变
在此面板中，你可以设置骨骼的基本变形属性。

Transform 变换¶

变换面板（编辑模式）。

在编辑模式下，你可以使用此面板来控制单个骨骼的位置和滚动（扭转）。

在姿态模式下，你只能设置主骨骼的位置，你现在可以设置旋转和缩放。

变换面板（姿态模式）。

Note

此模式仅适用于编辑姿势模式。

Transform Locks 变换锁定¶

变换锁定面板。

该面板仅在姿态模式显示，并允许你通过轴上每个骨骼上的轴限制位置，旋转和缩放。

Note

此模式仅在姿态模式下可用。

Inverse Kinematics 反向运动学¶

反向运动学面板。

这个面板控制当它链接在一个反向运动学的链时一个骨骼或一组骨骼的行为。

Note

此模式仅在姿态模式下可用。

Custom Properties 自定义属性¶

有关详细信息，请参阅自定义属性（ Custom Properties ）页面。
